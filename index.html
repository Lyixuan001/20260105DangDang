当当网图书数据分析项目

项目背景

本项目基于当当网图书销售数据，对销量、价格、折扣等指标进行整理和分析， 用于练习数据清洗、统计分析及结果可视化。

主要工作

  • 使用 Python 对原始 Excel 数据进行清洗与整理
  • 统计不同时间维度下的销量与折扣情况
  • 生成分析图表并输出结果文件

技术栈

  • Python
  • Pandas / NumPy
  • Matplotlib（数据可视化）

项目结构

  • *.py：数据处理与分析脚本
  • *.xlsx：原始及处理后的数据文件
  • *.png：分析结果图表
  • README.md：项目说明
